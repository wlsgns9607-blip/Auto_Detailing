# 📌 Project Overview

Auto Detailing은  
자동차 디테일링 서비스를 보다 전문적이고 신뢰감 있게  
보여주기 위해 제작한 웹 프로젝트입니다.

기존 자동차 관리 업체들의 복잡하고 올드한 UI 대신  
깔끔한 레이아웃과 직관적인 사용자 경험(UI/UX)을 중심으로  
브랜드형 웹사이트를 구현했습니다.

---

# 🎯 Project Goals

- 디테일링 업체의 브랜드 신뢰도 강화
- 사용자 중심의 직관적인 UI/UX 설계
- 반응형 웹 기반 모바일 최적화
- 서비스 소개 및 상담 접근성 향상
- 포트폴리오 중심의 비주얼 강조

---

# ✨ Main Features

## 🏠 메인 랜딩 페이지
- 브랜드 중심 Hero Section
- 서비스 핵심 소개
- 시각적 강조 레이아웃

## 🚘 디테일링 서비스 소개
- 차량 관리 서비스 구성
- 프리미엄 서비스 강조
- Before / After 이미지 영역

## 📱 반응형 웹
- 모바일 / 태블릿 / 데스크탑 대응
- Flex & Grid 기반 레이아웃

## 🎨 UI/UX 설계
- 미니멀 & 프리미엄 디자인
- 직관적인 사용자 흐름 구성
- 사용자 집중형 인터페이스
